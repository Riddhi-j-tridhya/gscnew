== Installation ==

1. Upload the "Contact Form 7 Database Manager Addon – CF7DBM" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Look for the ‘Submissions’ item in the Admin left menu, under the ‘Contact’ menu.
4. Enjoy!

5. If you already have some contacts saved via other plugins, navigate to "Contact" / "Submissions Import" in the Admin page to import them into CF7DBM.

== Frequently Asked Questions ==

= Does this plugin work with other form plugins? =

No. "Contact Form 7 Database Manager Addon – CF7DBM" is exclusively designed for Contact Form 7.

= How do I export the form data? =

Simply navigate to the plugin page "Contact/Submissions", select the desired form, and click on the "Export" button located on the right. This will download a CSV file containing submissions from the selected form.

= How can I import data from other plugins into CF7DBM? =

With CF7DBM, you can seamlessly import data stored by other plugins. Navigate to "Contact" / "Submissions Import" on the Admin page. If CF7DBM detects data from other plugins in your database, you will be provided with an option to import those submissions.
